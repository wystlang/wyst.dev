# Security policy

Please report suspected vulnerabilities through GitHub's **Report a
vulnerability** form on this repository's Security tab. Reports submitted there
are private to the Wyst maintainers; do not disclose an unpatched vulnerability
in a public issue.

Only the current production release at [wyst.dev](https://wyst.dev) is
supported. Include the affected URL, reproduction steps, impact, and any
suggested mitigation. Please avoid accessing data that is not your own or
disrupting the public service while testing.
